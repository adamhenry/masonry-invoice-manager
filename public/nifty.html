Nifty Corners

This is the popular Nifty Corners ported to jQuery.

How to use

You call the plugin like this: $("query").nifty(options)

For documentation on available options, please read the original documentation page of Nifty Corners Cube. Have fun!

Simple. Just Javascript and CSS.
